# FitWithPhu 🏋️‍♂️✨ (Cloudflare Pages)

เว็บไซต์สำหรับคำนวณค่าดัชนีมวลกาย (BMI) พร้อมคำแนะนำและโปรแกรมการออกกำลังกายที่เหมาะสมกับร่างกายของคุณ พัฒนาด้วยเทคโนโลยี Vanilla HTML, CSS และ JavaScript สำหรับเป็นเครื่องมือดูแลสุขภาพอย่างง่ายและปลอดภัยที่บ้าน

## 🚀 ฟีเจอร์หลัก (Key Features)

* **เครื่องคำนวณ BMI อัจฉริยะ:** ประเมินดัชนีมวลกายพร้อมแถบวัดสถานะและระดับความฟิต
* **คำแนะนำเฉพาะบุคคล:** แนะนำโภชนาการและการออกกำลังกายที่เหมาะกับผลลัพธ์ BMI ของคุณ
* **ระบบจับเวลาออกกำลังกาย (Workout Timer):** มีระบบจับเวลาและนับถอยหลังพร้อมเสียงเตือนสำหรับการฝึกแต่ละท่า
* **การออกแบบที่สวยงามและรองรับมือถือ (Responsive Web Design):** แสดงผลได้อย่างสวยงามในทุกอุปกรณ์

---

## 🛠️ วิธีการติดตั้งและรันโปรเจกต์ (Installation & Setup)

โปรเจกต์นี้ทำงานโดยใช้ Node.js เป็นเซิร์ฟเวอร์แบบ Static
หากต้องการจำลองระบบบนเครื่องโลคอล (localhost) ให้ปฏิบัติตามขั้นตอนต่อไปนี้:

### 1. โคลนคลังข้อมูล (Clone Repository)
```bash
git clone https://github.com/PhumiratPMTJ/website.git
cd website
```

### 2. รันเว็บเซิร์ฟเวอร์ (Run Server)
คุณสามารถสั่งรันเซิร์ฟเวอร์จำลองบนพอร์ต 3000 ได้ทันทีโดยไม่ต้องติดตั้งไลบรารีภายนอกเพิ่มเติม:
```bash
npm start
```
หรือรันผ่าน Node.js โดยตรง:
```bash
node server.js
```

หลังจากเริ่มเซิร์ฟเวอร์แล้ว ให้เปิดเบราว์เซอร์และเข้าไปที่ **[http://localhost:3000](http://localhost:3000)** เพื่อใช้งาน

## 📂 โครงสร้างโฟลเดอร์ (Project Structure)

* `index.html` - หน้าโครงสร้างหลักของเว็บไซต์
* `style.css` - สไตล์การตกแต่งและแอนิเมชันทั้งหมด (Vanilla CSS)
* `app.js` - โค้ดคำนวณ BMI และระบบควบคุมจับเวลา
* `server.js` - ไฟล์เซิร์ฟเวอร์ Node.js สำหรับพอร์ต 3000
* `package.json` - ไฟล์กำหนดสคริปต์และข้อมูลโปรเจกต์
* `assets/` - โฟลเดอร์เก็บภาพหรือทรัพยากรต่าง ๆ ที่ใช้ในระบบ
